Save Using a Registered Memo.

Benefits

With this shortcut, you can save a log by just tapping the Shortcut button — no typing required.
This makes recording quick and effortless.
You can also edit preset memo texts and place the shortcut on your Home Screen for easy access.

How to Set Up

Use Shortcuts app.
Step.1
Open the Settings app  → Shortcuts  →   enable Allow Untrusted Shortcuts.
Step.2
Download the shortcut from the link below.
Get shortcut from here (tap twice)
Then open the Shortcuts app and find “Record Time with Memo.”
Edit the text in “Enter what you want to record here” to customize the memo.

Step.3
When you run the shortcut for the first time, you’ll see a message:
“Allow Untrusted Shortcut?”
Tap Allow to enable it.
