Tips


Select from your own list and record

Useful hint

You can collect the notes you often record yourself in a list, and when you want to record, select from there and record.
Eliminates the need to write notes every time.

How to set

Use the "shortcut" app.
Step.1
Setting > Shortcuts, at "SHARING SECURITY", turn on "Allow Untrusted Shortcuts"

Step.2
Get a shortcut from the link below.
Get a shortcut from here

Find the shortcut for "Record time from list" from the "Shortcuts" app and edit the list.


Step.3
When you execute the shortcut for the first time, the message "Allow Untrusted Shortcuts?" is displayed. Click "Allow".


Record with registered memo

Useful hint

You can record quickly because you don't have to enter anything with one tap of the shortcut button.
It is more convenient to edit some shortcut memos and put them together on the home screen.

How to set

Use the "shortcut" app.
Step.1
Setting > Shortcuts, at "SHARING SECURITY", turn on "Allow Untrusted Shortcuts"

Step.2
Get a shortcut from the link below.
Get a shortcut from here
Find the shortcut for "Record time with memo" from the "Shortcuts" app, and edit "Enter what you want to record here" from Edit.

Step.3
When you execute the shortcut for the first time, the message "Allow Untrusted Shortcuts?" is displayed. Click "Allow".
