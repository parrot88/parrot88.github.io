Select from the list you created by your own to save it

Convenient point

You can put a list together from the log, which you often save it. When you save the log, you can select the save log from the list.
Then you don’t need to write memos every time when you log.

How to set

Use Shortcuts app.
Step.1
Go to Setting>Shortcuts, sharing security> Turn on (Green) Allow Untrusted Shortcuts

Step.2
"Get shortcuts" from the link below.
Get shortcut from here (tap twice)
From the shortcuts app, find the shortcut of "Record time from list". Edit the list which you want to save in the Edit.


Step.3
The first time you run the shortcut, you will see a message “Allow Untrusted Shortcuts?“ then press Allow.

Save from registered memo.

Convenient point

To save log with Shortcuts, just tap Shortcuts button once then you don’t need to input anything, so you can save quickly.
It is convenient to edit some shortcut memos and keep them on the home screen.

How to set

Use Shortcuts app.
Step.1
Go to Setting>Shortcuts, sharing security> Turn on (Green) Allow Untrusted Shortcuts

Step.2
Get a shortcut from the link below.
Get shortcut from here (tap twice)
Find the shortcut for "Record time with memo" from the Shortcuts app, and "Enter what you want to record here" from Edit.

Step.3
The first time you run the shortcut, you will see a message “Allow Untrusted Shortcuts?“ then press Allow.
